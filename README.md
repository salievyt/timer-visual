# 🌊 Djarvis Timer

**Красивый таймер обратного отсчёта с анимацией волны**

[![Flutter](https://img.shields.io/badge/Flutter-3.10+-02569B?style=for-the-badge&logo=flutter)](https://flutter.dev)
[![Dart](https://img.shields.io/badge/Dart-3.0+-0175C2?style=for-the-badge&logo=dart)](https://dart.dev)

---

## ✨ Особенности

| Фича | Описание |
|------|----------|
| 🌊 **Анимация волны** | Плавная анимация заполнения экрана водой |
| ⏱️ **Таймер** | Обратный отсчёт с круговым индикатором |
| ⚙️ **Настройка времени** | Установи любое время от 1 до 3600 секунд |
| 🎨 **Material Design 3** | Современный UI с тенями и анимациями |
| 🖱️ **Hover эффекты** | Кнопка увеличивается при наведении |

---

## 🎯 Использование

1. **Нажми на ⚙️** (шестерёнка) — откроется ввод времени
2. **Введи секунды** (1-3600) и нажми **OK**
3. **Нажми Start** — запустится таймер
4. Смотри как цифры меняются в реальном времени! 🔢
